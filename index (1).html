Web Conflict Predictor Extension

Overview

This software is a tool that can be used to predict conflicts online or on web platforms. Our main objective is to predict the conflict in the earlier stages and report it to moderators before the conflict gets out of hand and causes any sort of damage to people or community in general. This software can be used in practice to create early warnings for community moderators and to predict potential negative mobilizations as soon as any cross-linking post or message is created on their platform, and help moderators to curb the adverse effects of intercommunityconflicts. There are no particular constraints on operating systems for using the system.User documentation is provided along with the software.
